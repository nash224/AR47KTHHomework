
* === 프로젝트/파일/창 생성 및 열기 ===
* 새 프로젝트 생성하기 Ctrl + Shift + N
* 새 파일 추가하기 Ctrl + Shift + A
* 현재 문서 닫기 Ctrl + F4
* 기존 프로젝트 열기 Ctrl + Shift + O
* 기존 파일 열기 Ctrl + O
* 솔루션 탐색기 열기 Ctrl + Alt + L
*
* === 편집관련 단축키 Editing ===
* 자동완성 Ctrl + Space
* 자동 정렬하기 Ctrl + K + F
* 현재 라인 복사해서 아래라인에 붙이기 Ctrl + D
* 현재 라인 잘라내기 Ctrl + X, Ctrl + L
* 완전히 삭제하기 Ctrl + Shift + L
* 네모 박스 편집, 열모드 편집 Alt + 드래그 , Alt + Shift + ↑ / ↓
* 주석처리하기 Ctrl + K + C
* 주석해제 Ctrl + K + U
* 소문자변환 Ctrl + U
* 대문자변환 Ctrl + Shift + U
* 함수 접고 펴기 Ctrl + M + M
*
* === 커서 이동 관련 단축키 ===
* 현재 행에서 맨 앞으로 이동 Home
* 현재 행에서 맨 뒤로 이동 End
* 현재라인 전체선택 Shift + Home /  End
* 단어 단위로 커서 이동 Ctrl + ← / →
* 단어 단위로 드래그확장하기 Ctrl + Shift + ← / →
* 파일 맨 처음이나 맨 끝으로 이동하기 Ctrl + Home / End
* 특정 라인으로 이동하기, 행번호 이동 Ctrl + G
* 괄호 끝으로 이동 Ctrl + ]
* 괄호 범위 드래그 Ctrl + Shift + ]
* 줄 상하로 이동시키기 Alt + ↑ / ↓
*
* === 파일 이동 / 엿보기 관련 ===
* 함수 정의문으로 바로 이동 F12
* 함수 정의문 엿보기 Alt + F12
* 파일로 이동 Ctrl + Shift + G
*
* === 검색 / 찾기 / 바꾸기 ===
* 찾기 Ctrl + F
* 바꾸기 Ctrl + H
* 이름 일괄 변경 Ctrl + R + R
*
* === 빌드 / 컴파일 / 디버깅 관련 단축키 ===
* 솔루션 빌드 Ctrl + Shift + B
* 조사식 확인하기 Shift + F9
*
* == 그 외 단축키 ===
* 모든 참조 찾기 Shift + F12
* 헤더파일 자동 삽입 Alt + Enter / Ctrl + .
* 코드 감싸기 Ctrl + K + S
